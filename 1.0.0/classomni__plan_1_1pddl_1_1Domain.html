Detailed Description

Represents a PDDL domain with its requirements, types, predicates, and actions.

A PDDL domain defines the structure and rules of a planning problem, including the types of objects, predicates for describing states, and actions that can change the state. This class provides methods to build and manipulate domain components and generate PDDL output.
