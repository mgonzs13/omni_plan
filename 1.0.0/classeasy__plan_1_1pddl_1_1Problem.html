Detailed Description

Represents a PDDL problem with its objects, goals, and facts.

A PDDL problem defines a specific instance of a planning domain, including the objects that exist, the initial state (facts), and the goals that need to be achieved. This class provides methods to build and manipulate problem components and generate PDDL output.
