Detailed Description

Planner implementation using the VHPOP (Versatile Heuristic Partial Order Planner) algorithm.

This class provides automated planning capabilities using the VHPOP algorithm, which is particularly effective for temporal planning domains with durative actions. VHPOP can handle complex planning problems involving concurrent actions and temporal constraints.
