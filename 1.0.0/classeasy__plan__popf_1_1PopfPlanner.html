Detailed Description

Planner implementation using the POPF (Partial Order Planning with First-Order logic) algorithm.

This class provides automated planning capabilities using the POPF algorithm, which is particularly effective for temporal planning domains with durative actions. POPF can handle complex planning problems involving concurrent actions and temporal constraints.
