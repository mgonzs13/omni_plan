Detailed Description

Base class for automated planning algorithms.

This abstract base class defines the interface for planning components that solve PDDL planning problems. Planners implement various search algorithms to find sequences of actions that achieve the goals defined in the problem.

Generates a plan to solve a PDDL planning problem.

This pure virtual method must be implemented by derived classes to provide the specific planning algorithm. It takes PDDL domain and problem definitions and a map of available actions, then returns a plan that solves the problem or indicates that no solution was found.

Parameters

Returns
