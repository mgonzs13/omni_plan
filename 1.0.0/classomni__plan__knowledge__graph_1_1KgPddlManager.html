Detailed Description

PDDL manager implementation using a knowledge graph for state representation.

This class extends the base PddlManager to provide knowledge graph-based state management. It maintains the current world state in a knowledge graph and provides methods to query and update this state. The manager can optionally register callbacks to automatically update the state when the knowledge graph changes.
