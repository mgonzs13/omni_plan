Detailed Description

Planner implementation using the SMTP (Simple Temporal Problem) algorithm.

This class provides automated planning capabilities using the SMTP algorithm, which is particularly effective for temporal planning domains with durative actions. SMTP can handle complex planning problems involving concurrent actions and temporal constraints.
